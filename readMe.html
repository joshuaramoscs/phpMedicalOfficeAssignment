Medical Office Scheduler README Page

Medical Office Scheduler Logistics

Appointment hours: From 9:00 AM to 6:00 PM.

Possible appointment times: Max: Every quarter hour.

Default Appintment times: Established patients: 30 mins. New patients: 90 mins.

Drive Time: Each coordinate move in the X or Y-axis takes 2 minutes.

Medical Office Scheduler Warnings

If Patient's name is left blank, patient will not be scheduled.

If Patient's coordinate is left blank, default coordinate "A0" will be assigned.

If Patient's start time is set, but not their end time, then defailt appointment time will be set.

If desired start time is after desired end time, desired end time will be calculated to default appointment time.
